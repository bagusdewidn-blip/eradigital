Prototipe aplikasi ini dirancang sebagai solusi inovatif untuk mengatasi permasalahan pelayanan publik pada Dinas Kependudukan dan Pencatatan Sipil (Disdukcapil), khususnya terkait ketidakpastian waktu dan prosedur yang berbelit dalam penerbitan dokumen kependudukan.
ğŸ“‹ Tentang Proyek
Aplikasi ini menggabungkan pendekatan New Public Management (NPM) melalui digitalisasi sistem dan New Public Service (NPS) melalui fitur-fitur yang berorientasi pada pelayanan warga.
Filosofi C-A-K-A-P:
Cepat: Otomatisasi verifikasi dan pemangkasan birokrasi.
Akuntabel: Pelacakan (tracking) status dokumen secara real-time dan transparan.
Kolaboratif: Integrasi data antar lembaga (simulasi).
Adaptif: Menyesuaikan dengan teknologi terkini (Mobile-first design).
Proaktif: Memberikan notifikasi dan solusi alternatif (IKD) sebelum warga bertanya.
âœ¨ Fitur Utama
Sistem Pelacakan (Tracking) Real-Time
Warga dapat memasukkan Nomor Registrasi atau NIK untuk melihat posisi dokumen mereka (Pengajuan -> Verifikasi -> Biometrik -> Pencetakan).
Menjawab masalah "Ketidakpastian Waktu".
Identitas Kependudukan Digital (IKD)
Kartu digital dengan efek holografik modern.
Solusi alternatif (Virtualisasi Dokumen) ketika blanko fisik habis.
Notifikasi Proaktif
Popup notifikasi yang memberitahu warga mengenai status pengajuan atau saran aktivasi IKD tanpa perlu mengecek manual terus-menerus.
Simulasi Pengajuan
Fitur demo untuk membuat pengajuan baru dan melihat bagaimana sistem memproses data secara otomatis.
