Aplikasi ini menggabungkan pendekatan New Public Management (NPM) melalui digitalisasi sistem dan New Public Service (NPS) melalui fitur-fitur yang berorientasi pada pelayanan warga.

### Filosofi C-A-K-A-P:

### *Cepat*
Otomatisasi verifikasi dan pemangkasan birokrasi.

### *Akuntabel*: 
Pelacakan (tracking) status dokumen secara real-time dan transparan.

### *Kolaboratif*:
Integrasi data antar lembaga (simulasi).

### *Adaptif*:
Menyesuaikan dengan teknologi terkini (Mobile-first design).

### *Proaktif*: 
Memberikan notifikasi dan solusi alternatif (IKD) sebelum warga bertanya.

## âœ¨ Fitur Utama

### 1. Sistem Pelacakan (Tracking) Real-Time

Warga dapat memasukkan Nomor Registrasi atau NIK untuk melihat posisi dokumen mereka.

Alur: Pengajuan -> Verifikasi -> Biometrik -> Pencetakan.

Menjawab masalah "Ketidakpastian Waktu".

### 2. Identitas Kependudukan Digital (IKD)

Kartu digital dengan efek holografik modern.

Solusi alternatif (Virtualisasi Dokumen) ketika blanko fisik habis.

### 3. Notifikasi Proaktif & WhatsApp Otomatis

WhatsApp Simulation: Sistem secara otomatis "mengirimkan" notifikasi ke nomor WhatsApp pengguna (simulasi Toast notification).

In-App Notification: Notifikasi tersimpan dalam menu lonceng.

### 4. Simulasi Pengajuan

Fitur demo untuk membuat pengajuan baru dan melihat bagaimana sistem memproses data secara otomatis.
